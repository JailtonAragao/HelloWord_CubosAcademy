     - Iniciando o versionamento com o terminal do VScode.
       - Comandos:
         - git --version >> Retorna a versão atual do git instalado.
         - git init >> Inicializa o repositório na pasta do código.
         - git status >> Verificar estado do arquivo e retorna os pendentes a serem visionados.
         - git add . >> Add o arquivo a chave de versionamento.
         - git commit -m >> Add comentários no versionamento.
         - git branch -M >> Indica a ramificação.
         - git remote >> Endereça o arquivo ao repositório .
         - git push >> Enviar para o repositório.
